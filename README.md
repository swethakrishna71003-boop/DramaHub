# DramHub 🎬

## 📌 Description
DramHub is a web application where users can browse and explore dramas.  
It includes user authentication with secure login using JWT.

## 🚀 Features
- User Registration & Login
- JWT Authentication
- Protected Dashboard
- Responsive UI
- Search functionality (if you added)

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL
- Authentication:  JWT, bcrypt

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
3. install npm then Start server:
4. node server.js  then Open in browser: http://localhost:4000

   
## 🔐 Login Features
- Password hashing using bcrypt
- Token-based authentication (JWT)
- Protected routes
